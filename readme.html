Welcome to adispeak

What is Adispeak?

Adispeak is an extention of AdiIRC It's aim is to make IRC fully accessible with any screenreader. It uses the Tolk library to interface with screenreaders that are installed or sapi speech.


Building Adispeak from source

If you'd like to get AdiSpeak's source, you can find it Here.

If you install adispeak by downloading the official AdiIRC, Put everything in adispeak\bin\x64\release into the same folder as AdiIRC.exe once you download AdiIRC from the link above. Run the .exe, and choose file>plugins>load and choose adispeak.dll. Press OK, and everything should start speaking!

If you'd like to build AdiSpeak.dll from source, you need Visual Studio 2017 installed. Open the .sln file and it should open in Visual Studio, and press build.


Once AdiSpeak is running

Once you have AdiSpeak installed, either by getting the official client and adding AdiSpeak to it, or by getting one of the preconfigured releases, go to file>options, and go through each thing in the review, setting up AdiIRC how you'd like it. Once you've done that, you should be placed back into an edit box. From here, enter the standard IRC commands that you know to connect to an IRC server.


Commands

The following is the list of keys that AdiSpeak supports for controlling speech and history.

  • f3: Switch between SAPI speech and screenreader output. If using Sapi, to adjust rate/volume/voice, you must do it through speech settings in control panel.
  • F4: Disable and enable speech globally.
  • f5: Disable and enable speech for the currently focused window.
  • F6: Enter the history window. Once here, the following commands become active.
  • Arrow keys: move around the history buffer.
  • Space: repeat the last thing spoken.
  • Shift+C: Copy the currently selected line to the clipboard.
  • Escape: Close the history buffer and return to the edit field to input text.

If you'd rather not open the history buffer and want to keep the messages onscreen, you can use the following commands.

  • Alt+shift+up and down: Review the message history.
  • Control+shift+C: Copy the currently selected message to the clipboard.

Scripting

The following commands can be given to control the speech system.

  • /speak : Speak some text.
  • /braille : Braille some text.
  • /sout : Output text using the user's prefered method (speech or braille).
  • /sapion: Force sapi on.
  • /sapioff: Force screen reader.
  • /savespeech: Save your speech settings.
  • /loadspeech: Reload your speech settings.
  • $screenreader: Use in any script to get the name of the active screen reader.
  • $speech: Returns true if the user's screen reader supports speech.
  • $braille: Returns true if the user's screen reader supports braille.
  • $speaking: Returns true if the screen reader is currently speaking.
